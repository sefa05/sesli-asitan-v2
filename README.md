# Çamlıca Diş Kliniği – Sesli Asistan Paneli

Bu proje Next.js 14 (App Router) + TypeScript + Tailwind CSS kullanılarak hazırlanmış bir yönetim paneli şablonudur. Supabase ile kimlik doğrulama ve veritabanı entegrasyonu örnekleri içerir.

Kurulum adımları:

1. Node v18+ olduğundan emin olun.
2. Kopyalayın ve bağımlılıkları yükleyin:

```powershell
npm install
```

3. `.env.local` dosyasını oluşturun ve gerekli değişkenleri ekleyin (örnek: `.env.local.example`).
4. Geliştirme modunda çalıştırın:

```powershell
npm run dev
```

Supabase tabloları için örnek SQL dosyaları proje içindeki dökümana eklenmiştir. Vercel üzerine deploy ederken aynı environment değişkenlerini Vercel dashboard'una ekleyin.
